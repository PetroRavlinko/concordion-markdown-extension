Markdown Syntax Extensions

The underlying Markdown parser is Pegdown, which allows for an extended Markdown syntax.

We are already using the TABLES and STRIKETHRU extensions to enable our syntax.

Should you wish to add further Pegdown extensions, call the method withPegdownExtensions(int options) where options is a bitmask of the required extensions.

No extra extensions

Without additional extensions, the following are translated as-is.

Example

Markdown Resulting HTML
[[not a wikilink]] [[not a wikilink]]
"pretty quotes" "pretty quotes"

With WIKILINK, SMARTS and QUOTES extensions

Given the method MarkdownExtension.withPegdownExtensions(int options) has been called with options set to org.pegdown.Extensions.WIKILINKS | org.pegdown.Extensions.QUOTES, the following are now translated as shown:

Example

Markdown Resulting HTML
[[wikilink]] <a href="./wikilink.html">wikilink</a>
"pretty quotes" &ldquo;pretty quotes&rdquo;
